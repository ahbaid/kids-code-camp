# Lesson 003 - Types

## int and str
~~~~python
>>> age=48
>>> name='Ahbaid'
>>> type(age)
<class 'int'>
>>> type(name)
<class 'str'>
~~~~

~~~~python
>>> 3/2

1.5

3.142857142857143
~~~~
